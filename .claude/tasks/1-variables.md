# [1] Переменные пользователя

## Задача
Объявить все настраиваемые переменные скрипта, сгруппированные по назначению, в одном блоке в начале файла.

## Цель
Пользователь редактирует только блок переменных перед загрузкой скрипта на сервер — остальной код не трогает.

## Обоснование
Единый блок переменных устраняет необходимость в отдельных скриптах для каждой ноды.

## Предусловия
Нет — это первая задача, скрипт начинается с этого блока.

## Результат
- Блок переменных объявлен в начале скрипта и доступен для всех последующих шагов

## Детали

### Переключатели
| Переменная | Значения | По умолчанию | Описание |
|---|---|---|---|
| `INSTALL_DOTNET` | `0`/`1` | `0` | `1` — установить .NET 10 backend (:8081). `0` — стандартная заглушка Caddy |
| `INSTALL_TG` | `0`/`1` | `0` | `1` — установить Telegram Local Server (:8082). Требует `TG_API_ID` и `TG_API_HASH` |
| `ENABLE_IPV6` | `0`/`1` | `0` | `0` — отключить IPv6 через sysctl |

### Идентификация ноды
| Переменная | Описание |
|---|---|
| `NODE_HOSTNAME` | Hostname сервера (применяется через `hostnamectl`) |
| `NODE_DOMAIN` | Полный домен ноды; на него выписывается LE-сертификат через Caddy ACME (:80) |

### Цепочка
| Переменная | Значения | Описание |
|---|---|---|
| `BACK_HOP` | домен/IP | Адрес предыдущей ноды (или nodeA напрямую). Для nodeA: `localhost:8081` если `INSTALL_DOTNET=1`, иначе пусто |

### Telegram Local Server
| Переменная | Описание |
|---|---|
| `TG_API_ID` | API ID приложения (только если `INSTALL_TG=1`) |
| `TG_API_HASH` | API Hash приложения (только если `INSTALL_TG=1`) |

### fail2ban
| Переменная | По умолчанию | Описание |
|---|---|---|
| `FAIL2BAN_MAXRETRY` | `15` | Максимум неудачных попыток до бана |
| `FAIL2BAN_FINDTIME` | `300` | Окно обнаружения, секунды |
| `FAIL2BAN_BANTIME` | `-1` | Длительность бана (`-1` = навсегда) |

### Кредентиалы
| Переменная | Формат | Генерация | Описание |
|---|---|---|---|
| `FP_CREDS` | `("user:pass" ...)` | `openssl rand -base64 12 \| tr -d '=/+'` | Кредентиалы forward proxy (HTTP CONNECT). Bash-массив |
| `FP_EXTRA` | целое | — | Количество автогенерируемых дополнительных FP-пользователей |
| `TELEMT_CREDS` | `("alias:32hex" ...)` | `openssl rand -hex 16` | MTProto-секреты. Ровно 32 hex-символа. Bash-массив |
| `TELEMT_EXTRA` | целое | — | Количество автогенерируемых telemt-пользователей |
